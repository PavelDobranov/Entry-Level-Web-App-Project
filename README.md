### How to run
- Install [nodejs](http://nodejs.org/) - preferably v.6.10.0 or higher

#### Web api
- Navigate the terminal to the `/server` folder

    - `npm install`

    - `npm start build`

    - `npm start dev`

#### Angular app
- Navigate the terminal to the `/client` folder

    - `npm install`

    - `npm start build`

    - `npm start dev`
